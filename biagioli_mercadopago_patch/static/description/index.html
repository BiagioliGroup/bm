Parche oficial para integrar correctamente Mercado Pago en Odoo

Solución directa a los errores del módulo abandonado de Mercado Pago

¿Por qué necesitas este módulo?

El módulo oficial de Mercado Pago para Odoo fue abandonado por los desarrolladores

Este parche soluciona los errores más comunes como:

    li> Posibilidad de Ofrecer Cuotas en las ventas (El pluguin oficial de Odoo solamente deja ofrecer 1 sola cuota al vender )
  • Redirecciones inválidas (auto_return)
  • URLs malformadas con HTTP en lugar de HTTPS
  • Campos requeridos faltantes que disminuyen la tasa de aprobación

Está diseñado para funcionar sobre el módulo payment_mercado_pago original sin romper compatibilidad con otros pagos.

  • ✔ Redirección segura con HTTPS
  • ✔ Mejora de tasa de aprobación con payload completo
  • ✔ Compatible con Odoo 16, 17 y 18
  • ✔ Mínima configuración: solo instalar y listo

Configuración de cuotas desde Mercado Pago

Importante:
Para poder configurar las condiciones de venta y los costos asociados a ofrecer cuotas con o sin interés, es necesario hacerlo directamente desde el panel de Mercado Pago:

  • Ruta: Tu negocio → Costos y cuotas → Opción: Checkout
  • Desde allí podrás habilitar o deshabilitar opciones como Cuota Simple o Cuotas sin interés, y definir quién absorbe el costo.
